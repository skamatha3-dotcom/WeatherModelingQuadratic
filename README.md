# Weather Modeling Using Quadratic Equations

## Features
- v1: Hand-coded variables
- v2: Keyboard input for single set
- v3: File input for multiple sets
- v4: File input + save output to file

## How to Run
1. Compile in C: `gcc weather_roots_vX.c -o weather_roots_vX -lm`  
2. Run: `./weather_roots_vX`  
3. For file input, ensure `input.txt` is in the same folder
